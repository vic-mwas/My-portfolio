Masakani – Real Estate Platform

Masakani is a full-stack real estate platform built with Flutter, Firebase, and Cloud Functions. It includes a Web App, Android App, and Admin Dashboard with role-based access and secure payment workflows.

  • Flutter (Web & Android)
  • Firebase Firestore & Storage
  • Node.js & TypeScript (Cloud Functions)
  • Payment-gated content unlocking
  • Role-based access (User, Agent, Admin)
  • API testing using Postman

Web App: https://masakani.app
Android App: Google Play Closed Testing (Early Access)
